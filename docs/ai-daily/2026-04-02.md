### 2. Anthropic Claude Code 源码意外泄露，震动 AI 行业

Anthropic 确认其 AI 编码工具 **Claude Code** 的完整源码在 npm 包 v2.1.88 中意外泄露——一个 59.8 MB 的 JavaScript source map 文件被误发布到公共 npm 仓库。

泄露内容揭示：
- Claude Code 年化收入已达 **25 亿美元** ARR
- 代码中引用了未发布的 **Claude Mythos** 和 **Capybara** 模型
- Anthropic 自述 Mythos "在网络安全能力上远超任何现有模型"
- 这是继上周 Mythos 博客草稿泄露后的**第二次安全事故**

Anthropic 已发出超 8000 条 DMCA 版权删除通知，试图控制传播。事件发生在 3 月 31 日，恰逢愚人节前夕，引发"是否是 PR 策略"的讨论。

### 8. 维基百科通过首个 LLM 禁令：44:2 压倒性票数
